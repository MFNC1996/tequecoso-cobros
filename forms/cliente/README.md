# Formularios de cliente

Esta carpeta contiene los formularios relacionados con la gestión de clientes dentro de la solución.

## Contenido
- `registrar-cliente.form`: formulario utilizado para el proceso de registro de clientes.

## Relación con el proyecto
Este formulario se asocia al proceso BPMN de registro de clientes y permite capturar la información necesaria para iniciar y completar dicha tarea en Flowable.
