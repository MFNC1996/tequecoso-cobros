# Formularios de proveedor

Esta carpeta contiene los formularios relacionados con la gestión de proveedores dentro de la solución.

## Contenido
- `registrar-proveedor.form`: formulario utilizado para el proceso de registro de proveedores.

## Relación con el proyecto
Este formulario se asocia al proceso BPMN de registro de proveedores y permite capturar la información requerida para crear y gestionar proveedores en Flowable.
