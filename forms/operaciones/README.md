# Formularios de operaciones

Esta carpeta contiene los formularios relacionados con las operaciones principales del negocio, como cobros y pagos.

## Contenido
- `cobro-clientes.form`: formulario utilizado en el proceso de cobro a clientes.
- `pago-proveedor.form`: formulario utilizado en el proceso de pago a proveedores.

## Relación con el proyecto
Estos formularios se asocian a los procesos BPMN de operaciones del negocio y permiten registrar la información necesaria para ejecutar cobros y pagos dentro de la solución implementada en Flowable.
