Tequecoso Cobros
Frontend visual para clientes y cobros
Conectado a Spring Boot + Flowable

Interfaz visual para registrar clientes y cobros

Los clientes se guardan directo en localhost:8080. Los cobros ahora inician el proceso BPMN cobroAClientes en Flowable (localhost:8090), que a su vez llama automáticamente al backend.

Resumen técnico

Backendhttp://localhost:8080
Clientes / Cobrosvía proceso Flowable
Proveedores / Pagosvía proceso Flowable
Flowablehttp://localhost:8090

Si el navegador bloquea las peticiones por CORS, habrá que habilitarlo en el backend (ya hecho) y verificar que Flowable también lo permita.
